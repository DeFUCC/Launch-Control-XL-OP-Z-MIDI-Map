# Launch-Control-XL-OP-Z-MIDI-Map
 
 MIDI mappings for controlling the teenage engineering op-z with a Novation Launch Control XL

## Flow

All 8 user pages are used

1. Overview of all 8 tracks in stems
2. Kick and snare
3. Hihat and perc
4. Motion
5. Bass
6. Lead
7. Arpeggio
8. Chords
